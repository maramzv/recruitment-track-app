# Recruitment Track App

## Overview

Recruitment Track App is an application designed to help recruiting coordinators (like Priya) organize and manage the recruitment process by uploading an existing spreadsheet or manually entering candidate information (name, recruitment stage, last follow-up date, department, hiring manager, etc.) and getting an easy to use dashboard that shows her where everyone is in the hiring process and who needs her attention first by last contact. 


## MVP Core Problem and Solution

Priya already has a working spreadsheet but struggles to create an effective follow-up reminder system that shows which candidates have been waiting the longest and need her attention first. This pain point keeps her up at night the most.

The solution is a tool that helps the recruiting coordinator prevent candidates from going cold by automatically tracking follow-ups and highlighting which candidates require immediate attention.

## Features

- Track candidates who applied for jobs
- Store candidate information
- Monitor hiring process stages
- Keep notes and track important dates
